# FirstBlood
一个基于SpringBoot+Vue3实现的前后端分离后台管理系统。在学习的过程中让自己也成长了很多，踩了很多坑但也收获到了很多。

## 设计知识点

前端：Vue3、Element-Plus、Echarts、Axios、Vue-Router

后端：SpringBoot、MyBatis-Plus

JDK：1.8

MySQL：8.0

其他相关：Hutool工具、JWT等。
